Sample code demonstrating automatic continuous conversions for U545 ADC1 with GPDMA using "Standard Request Mode". <br>
ADC1 : VREFINT, IN1, IN2, Temperature Sensor <br>
ADC2 : IN3, IN4 <br>
ADC calibration done at start-up <br>
<br>
TESTED USING NUCLEO-U545RE-Q <br>
<br>
